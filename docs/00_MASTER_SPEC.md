# Newtheria 게임 기획 마스터 문서

- 버전: **ver.0.0.19(260329-별자리라인연출추가)**
- 인코딩: UTF-8
- 상태: Draft (Refactored)

## 0. 목적
이 문서는 상세 규칙을 모두 복제하지 않고, **시스템 간 연결과 우선순위**를 정의하는 최상위 기준 문서입니다.

## 1. 프로젝트 스냅샷
- 장르: 타일 전략 시뮬레이션 + 턴제 전투 RPG + 장수전
- 월드 크기: 200x200 타일
- 좌표 체계: (x, y, z)
- 현재 구현 우선순위: 지상계(z=0)

## 2. 핵심 원칙
1. **대칭 원칙**: 플레이어/ NPC는 동일 규격의 행동 규칙을 공유한다.
2. **클래스리스 원칙**: 고정 직업 없이 스탯/장비/행동 이력 기반으로 성장한다.
3. **문서 단일 책임**: 상세 규칙은 해당 도메인 문서에서만 관리한다.
4. **변경 추적**: 규칙 변경 시 `99_변경이력.md`를 반드시 갱신한다.
5. **확장 안전성**: 모드 스크립트는 샌드박스/권한 제한 환경에서만 실행한다.

## 3. 도메인 문서 맵
- 월드/타일: `01_월드규칙.md`
- 정착지/내정: `02_정착지규칙.md`
- 국가/수도: `03_국가수도규칙.md`
- 전투 연동: `04_전투연동규칙.md`
- 캐릭터 프로필: `05_캐릭터프로필규칙.md`
- 달력 시스템: `06_달력규칙.md`
- 캐릭터 렌더링: `07_캐릭터레이어렌더링규칙.md`
- 카드 선택 연출: `08_카드선택연출규칙.md`
- 시스템 의존관계 맵: `09_시스템연계맵.md`
- 저장/불러오기: `10_저장불러오기규칙.md`

- 변경 이력: `99_변경이력.md`

## 4. 시스템 인터페이스(요약)
### 4.1 월드 ↔ 전투
- 조우/침공 이벤트 발생 시 전투 진입
- 전투 결과는 치안/영향권/정착지 상태에 반영

### 4.2 달력 ↔ 카드/이벤트
- 절기/월/주 상태가 이벤트 테이블 및 카드 풀에 가중치로 적용 가능

### 4.3 캐릭터 프로필 ↔ 렌더링
- 프로필(종족/체형/외형 태그)이 레이어 선택/틴트 팔레트 기본값에 영향

### 4.4 저장/불러오기 ↔ 전 시스템
- 월드/정착지/국가/캐릭터/달력/카드 상태를 저장 슬롯으로 직렬화
- 저장 포맷 버전 관리로 향후 마이그레이션 지원

### 4.5 모드 ↔ 코어 시스템
- 모드는 훅(Hook) 기반으로 행동/계산 로직을 확장
- 코어 데이터 접근은 권한 범위 내 API로 제한

### 4.6 시작 화면 ↔ 내비게이션/연출
- 타이틀/카드 선택지/버전 표기를 메인 진입 UI로 제공
- 카드 선택 시 배경 별(glow star) 연출이 외곽으로 확산되며 화면 전환
- 배경에 별자리 라인(점 연결) 드로우/페이드 연출을 주기적으로 표시
- 버전 텍스트 클릭 시 카드 셔플 이스터에그 실행

## 5. 문서 우선순위
