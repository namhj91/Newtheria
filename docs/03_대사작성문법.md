# 대사 CSV 작성 문법 가이드

> 대상: 기획자/작가/비전공자  
> 인코딩: **UTF-8**

이 문서는 `assets/js/dialogue_template.js` 기준으로, **실제로 동작하는 문법만** 정리합니다.

---

## 1) 기본 헤더 (권장 단일 스키마)

```csv
태그,앵커,화자,대사,배경,그림,그림애니메이션,스탠딩,bgm,bgs,voice,sfx,자동넘김,화면효과,카메라,ui,진동,파티클,조명,씬전환,변수,조건,효과,종료,선택지,히든선택지
```

- `태그`: 대화 묶음 식별자 (`intro`, `chapter1`, `quest_shop` 등)
- `앵커`: 태그 내부 진입점 (`start`, `choice`, `end` 등)
- `화자`: 이름 표시
- `대사`: 본문 텍스트 (`{{변수명}}` 치환 지원)
- `배경`: 이미지 경로/URL
- `그림`: 캐릭터 스탠딩/컷인 이미지 경로/URL
- `그림애니메이션`: `fade`, `float`, `zoom-in`, `shake` 프리셋
- `스탠딩`: 좌/우 캐릭터 배치 + 화자 하이라이트 + 간단 이동/방향 제어
- `bgm`: 배경 음악 제어
- `bgs`: 환경음 제어
- `voice`: 대사 보이스 제어
- `sfx`: 원샷 효과음 제어
- `자동넘김`: 선택지가 없는 줄에서 ms 단위 자동 진행 (예: `1200`)
- `화면효과`: `flash`, `shake`, `dim`
- `카메라`: `pan(x,y,ms)`, `zoom(scale,ms)`, `reset`
- `ui`: `hud=off;name=off;choices=lock;skip=off`
- `진동`: `light`, `medium`, `heavy`, `custom(40,80,40)`
- `파티클`: `snow:on`, `ember:on`, `off`
- `조명`: `tint(#88aaff,0.24,360)`, `off`
- `씬전환`: `fade-black(500)`, `fade-white(360)`
- `변수`: 현재 줄에서 주입할 변수 (`key=value` 묶음)
- `조건`: 줄 노출 조건식
- `효과`: 줄 진입 시 적용할 변수 변경식
- `종료`: `예/true/1/on/end/종료` 계열 입력 시 진행 정지
- `선택지`: 기본 분기 카드 목록
- `히든선택지`: 카드 전부 버리기(discard) 후 보여줄 분기 카드

## 1-1) 미디어 문법 (그림 / BGM / BGS / Voice / SFX)

### 그림 (`그림`, `그림애니메이션`)

- `그림`: 이미지 URL/경로를 넣으면 배경 위에 표시됩니다.
- `그림애니메이션`: 아래 키워드 중 하나를 넣습니다.
  - `fade`
  - `float`
  - `zoom-in`
  - `shake`

### 스탠딩 (`스탠딩`)

```text
좌=assets/img/asteria.png|아스테리아|move(12,0,280)|right;우=assets/img/player.png|순례자|move(-10,0,280)|auto;화자=좌
```

- 기본 형식: `좌=URL|이름|모션|방향;우=URL|이름|모션|방향;화자=좌|우`
- 슬롯 키
  - `좌`, `left`, `l`
  - `우`, `right`, `r`
- `이름`: `화자` 컬럼과 자동 매칭되어 발화자를 하이라이트(딤 해제)합니다.
- `모션`
  - `float`, `shake`
  - `move(x,y,ms)` (대략적인 위치 이동)
- `방향`
  - `auto`(기본): **모든 일러스트가 오른쪽을 보도록 그렸다는 가정**에서 우측 슬롯만 자동 반전해 서로 마주 보게 처리
  - `flip`, `left`: 강제 좌우반전
  - `right`, `unflip`: 반전 해제
- 전체 클리어: `off`, `clear`, `reset`, `none`

### 오디오 (`bgm`, `bgs`, `voice`, `sfx`)

아래 3가지 문법을 공통 지원합니다.

1) 정지
```text
stop
```

2) 상세 재생
```text
play|assets/audio/intro_bgm.mp3|0.7|true
```
- 형식: `action|url|volume|loop`
- `volume`: 0~1
- `loop`: `true/false` (또는 `예/아니오`)

3) 간단 재생(축약)
```text
assets/audio/voice_001.ogg
```
- URL만 쓰면 자동으로 재생으로 해석됩니다.
- `bgm`, `bgs`는 기본 loop=true, `voice`는 기본 loop=false 입니다.
- `sfx`는 기본 loop=false이며, 같은 줄 재진입 시 처음부터 다시 재생됩니다.

### 비대사 연출 (`자동넘김`, `화면효과`)

```text
자동넘김: 1000
화면효과: shake
```

- `자동넘김`: 선택지/종료가 아닌 줄에서 지정 시간(ms) 후 자동으로 다음 줄 진행
- `화면효과`:
  - `flash`: 순간 플래시
  - `shake`: 화면 흔들림
  - `dim`: 화면 암전 강조

### 시스템 연출 확장 (`카메라`, `ui`, `진동`, `파티클`, `조명`, `씬전환`)

```text
카메라: pan(12,-6,400);zoom(1.06,320)
ui: hud=on;name=on;choices=lock
진동: medium
파티클: snow:on
조명: tint(#88aaff,0.22,360)
씬전환: fade-black(500)
```

- `카메라`
  - `pan(x,y,ms)`: 화면 기준 이동
  - `zoom(scale,ms)`: 확대/축소
  - `reset`: 기본 위치/배율로 복귀
- `ui`
  - `hud=off|on`, `name=off|on`, `skip=off|on`
  - `choices=lock`으로 선택지 클릭 잠금
- `진동`
  - 프리셋: `light`, `medium`, `heavy`
  - 커스텀: `custom(40,80,40)` (ms 배열)
- `파티클`
  - `snow:on`, `ember:on`, `off`
- `조명`
  - `tint(색상,강도,지속시간)` 예: `tint(#88aaff,0.22,360)`
  - `off`
- `씬전환`
  - `fade-black(500)`
  - `fade-white(360)`

## 2) 선택지 문법

### 2-1. 기본 형태

```text
라벨 => scene#anchor
```

예시:

```text
도움을 요청한다 => intro#help
```

### 2-2. 앵커 생략

```text
라벨 => scene
```

- 앵커를 생략하면 해당 scene의 시작 지점으로 점프합니다.

### 2-3. 여러 선택지 연결

`||` 로 연결합니다.

```text
도움을 요청한다 => intro#help || 오늘은 물러난다 => intro#end
```

### 2-4. 선택지 메타(조건/효과)

```text
라벨 => scene#anchor [조건:신뢰도>=4] [효과:신뢰도+1;별조각+1]
```

- `[조건: ...]`: 이 선택지 자체의 노출 조건
- `[효과: ...]`: 선택 순간 적용되는 상태 변화

## 3) 변수 문법 (`변수` 컬럼)

### 3-1. 기본

```text
playerName=순례자;신뢰도=2;faction=은하수
```

### 3-2. 구분자

아래 구분자를 모두 지원합니다.

- `;`
- `,`
- 줄바꿈

### 3-3. 값 타입 자동 판별

- 숫자: `10`, `3.14`
- 불리언: `true`, `false`, `yes/no`, `예/아니오`, `on/off`
- 그 외: 문자열

> 팁: 처음 줄(`start`)에서 기본 변수를 세팅하면 이후 줄에서도 누적 활용하기 쉽습니다.

---

## 4) 조건 문법 (`조건` 컬럼)

### 4-1. 비교 연산자

- `>=`, `<=`, `>`, `<`, `==`, `!=`

### 4-2. 논리 연산

- 한글: `그리고`, `또는`
- 영문: `AND`, `OR`
- 부정: `NOT(...)`

### 4-3. 예시

```text
신뢰도>=3
신뢰도>=3 그리고 별조각>=1
신뢰도>=3 AND 별조각>=1
NOT(경계태세==true)
NOT(피로도>5) OR 별조각>=2
```

> 팁: 복합식은 괄호를 써서 의도를 명확히 하세요.

---

## 5) 효과 문법 (`효과` 컬럼)

효과는 한 셀에 여러 개 넣을 수 있으며 `;`, `,`, 줄바꿈으로 구분됩니다.

- 증가: `신뢰도+2` 또는 `신뢰도+=2`
- 감소: `별조각-1` 또는 `별조각-=1`
- 대입: `직업=기록관`
- 토글: `경계태세=!`

예시:

```text
신뢰도+1;별조각-1;경계태세=!
```

## 6) 템플릿 치환 (`{{변수명}}`)

대사/선택지 라벨에서 사용 가능합니다.

```text
{{playerName}} 님, 현재 신뢰도는 {{신뢰도}}입니다.
```

- 없는 변수 키는 원문 토큰(`{{없는키}}`)을 유지하므로 디버깅에 유리합니다.

---

## 7) 종료 문법 (`종료` 컬럼)

아래 값들은 종료로 인식됩니다.

- `예`, `true`, `1`, `yes`, `on`, `end`, `stop`, `종료`

종료 줄에서는 다음 진행(`▼`/클릭)이 막히므로, 보통 마지막 연출 줄에 사용합니다.

---

## 8) 변형/확장 사용 패턴 (실무 응용)

### 8-1. 퀘스트 상태 머신

- `quest_stage=튜토리얼` 세팅
- 조건: `quest_stage==튜토리얼`
- 효과: `quest_stage=첫임무`

이 방식으로 **문자열 상태 전이**를 구성할 수 있습니다.

### 8-2. 상점/재화 소비

- 조건: `gold>=30`
- 선택지 효과: `gold-=30;key_item=true`

**구매 가능/불가**를 CSV만으로 분기할 수 있습니다.

### 8-3. 신뢰도 기반 히든 루트

- 기본 선택지는 항상 노출
- `히든선택지`에 `[조건:신뢰도>=6]`

카드 discard 연출과 합쳐 숨겨진 경로를 자연스럽게 만들 수 있습니다.

### 8-4. 전투/이벤트 트리거 플래그

- 효과: `battle_ready=true`, `event_seen=true`
- 이후 조건: `battle_ready==true`

외부 시스템 연동 전에 **내부 플로우 테스트용 플래그**로 매우 유용합니다.

## 10) 작성 체크리스트

1. `태그/앵커` 네이밍 규칙을 팀 단위로 고정했는가?
2. `조건`은 괄호/연산자를 명확히 써서 오해 여지를 줄였는가?
3. `효과`가 의도대로 누적되는지 테스트 패널 변수 추적으로 검증했는가?
4. 종료 줄(`종료=예`) 이후 분기/진행이 막히는 동작을 확인했는가?
5. 샘플 CSV를 템플릿으로 복사해 새 시나리오를 시작했는가?
