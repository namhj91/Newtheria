# 02. 정착지 규칙

- 버전: **ver.0.0.21(260329-인덱스진입페이지추가)**

## 1) 정착지 단계
1. 야영지
2. 촌락
3. 마을
4. 도시
5. 성

## 2) 정착지 레벨
- 모든 단계는 정착지 레벨을 가진다.
- 승급은 `최소 레벨 + 단계별 조건`을 동시에 만족해야 가능하다.

### 2.1 승급 규칙(기본)
- 야영지 → 촌락: Lv3 + 인구수 조건
- 촌락 → 마을: Lv5 + 식량/치안 조건
- 마을 → 도시: Lv8 + 개발도/상업력 조건
- 도시 → 성: Lv12 + 방어력/행정력/국가 승인 조건

## 3) 단계별 스탯 적용 규칙
### 3.1 야영지 (특수)
- 인구수만 보유한다.
- 치안/개발도/생산력 등 일반 스탯은 비활성 처리한다.

### 3.2 촌락~성 (일반)
- 아래 핵심/보조 스탯 체계를 사용한다.

## 4) 핵심 스탯(플레이어 표시)
- 인구수
- 치안
- 개발도
- 식량
- 식수원
- 생산력
- 상업력
- 방어력
- 행복도
- 충성도

## 5) 보조 스탯(내부 계산)
- 청결도
- 위생
- 주거율
- 행정력
- 보급력
- 기술력
- 외교력
- 오염도
- 부패도
- 질서도
- 재난대응력
- 문화력
- 신앙력

## 6) 운용 원칙
- 핵심 스탯 중심으로 UI를 단순화
- 보조 스탯은 이벤트/공식 계산의 기반으로 사용
- 야영지는 개척 전초기지 개념으로 경량 관리한다
