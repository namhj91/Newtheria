# 06. 달력 규칙

- 버전: **ver.0.0.20(260329-시작화면HTML문서추가)**

## 1) 개요
- 달력은 계절(4), 월(12), 주(4) 구조를 사용한다.
- 월드 이벤트, 자원 생산, 전투 보정, 제례/축제 트리거의 기준 시간축으로 사용한다.

## 2) 절기(계절) 규칙
- 3~5월: **엘다나스의 절기(봄)**
  - 테마: 정령 / 생명, 개화, 신록
- 6~8월: **드라칸의 절기(여름)**
  - 테마: 드래곤 / 열기, 투쟁, 생명력
- 9~11월: **두르간의 절기(가을)**
  - 테마: 거인 / 수확, 대지, 제련
- 12~2월: **아르케의 절기(겨울)**
  - 테마: 천사 / 심판, 지혜, 침묵

## 3) 월 이름/룬 테마
1. 로고스 - 진리와 언령의 룬
2. 엔트로 - 혼돈과 소멸의 룬
3. 피오라 - 생명과 치유의 룬
4. 실루아 - 정령과 환영의 룬
5. 아스트라 - 별과 예지의 룬
6. 이그니스 - 기원염과 파괴의 룬
7. 볼바르 - 뇌우와 기상의 룬
8. 테라크 - 용혈과 변이의 룬
9. 가란 - 대지와 진동의 룬
10. 오리할 - 연금과 결속의 룬
11. 모리아 - 수호와 봉인의 룬
12. 아르카 - 순수 마나와 비전의 룬

## 4) 주차 이름/테마
1. 태동(胎動) - 마력이 막 깨어나는 주
2. 충만(充滿) - 마력이 가득 차오르는 주
3. 쇠퇴(衰退) - 마력이 서서히 기우는 주
4. 심연(深淵) - 마력이 가라앉아 휴식하는 주

## 5) 데이터 구조 제안
- `dateObj.year`
- `dateObj.month` (1~12)
- `dateObj.week` (1~4)
- `dateObj.day` (옵션)

## 6) 달력 정보 반환 규격(기획)
- `seasonName`, `seasonDesc`
- `monthName`, `monthDesc`
- `weekName`, `weekDesc`

## 7) 시스템 연동 예시
- 절기 버프: 봄(치유+), 여름(공격+), 가을(생산+), 겨울(저항+)
- 월 룬 효과: 해당 월에 특정 스킬군 강화
- 주차 변화: 심연 주차에 마나 회복/획득량 변동
