# 대사 CSV 작성 문법 가이드

> 대상: 기획자/작가/비전공자  
> 인코딩: **UTF-8**

## 1) 기본 헤더

```csv
태그,앵커,화자,대사,배경,변수,조건,효과,종료,선택지,히든선택지
```

- `태그`: 대화 묶음 이름(예: `intro`, `chapter1`)
- `앵커`: 대화 위치 이름(예: `start`, `end`)
- `화자`: 화면에 표시될 이름
- `대사`: 본문 텍스트 (`{{변수명}}` 치환 가능)
- `배경`: 배경 이미지 경로/URL
- `변수`: `key=value` 목록 (`;`, `,`, 줄바꿈으로 구분)
- `조건`: 줄 노출 조건
- `효과`: 줄 진입 시 변수 변경
- `종료`: `예/아니오` (`예`면 그 줄에서 진행 멈춤)
- `선택지`: 기본 선택지 목록
- `히든선택지`: 숨김 선택지 목록

---

## 2) 선택지 문법

### 2-1. 기본

```text
라벨 => scene#anchor
```

예시:

```text
도움을 요청한다 => intro#help
```

### 2-2. 여러 개 선택지

`||` 로 이어서 작성합니다.

```text
도움을 요청한다 => intro#help || 오늘은 물러난다 => intro#end
```

### 2-3. 선택지 메타(조건/효과)

```text
라벨 => scene#anchor [조건:신뢰도>=4] [효과:신뢰도+1]
```

---

## 3) 조건 문법

## 비교 연산자
- `>=`, `<=`, `>`, `<`, `==`, `!=`

## 논리 연산
- `그리고` / `또는`
- `AND` / `OR`
- `NOT(...)`

예시:

```text
신뢰도>=3
신뢰도>=3 그리고 별조각>=1
NOT(피로도>5)
```

---

## 4) 효과 문법

효과는 한 셀에 여러 개 작성할 수 있고 `;`, `,`, 줄바꿈으로 구분됩니다.

- 증가: `신뢰도+2` 또는 `신뢰도+=2`
- 감소: `별조각-1` 또는 `별조각-=1`
- 대입: `직업=순찰자`
- 토글: `경계태세=!`

예시:

```text
신뢰도+1;별조각-1
```

---

## 5) 변수 치환 문법

대사/선택지 라벨 안에서 `{{변수명}}` 형태로 사용합니다.

예시:

```text
{{playerName}} 님, 준비되셨어요?
현재 신뢰도는 {{신뢰도}}예요.
```

## 6) 최소 예시 템플릿

```csv
태그,앵커,화자,대사,배경,변수,조건,효과,종료,선택지,히든선택지
intro,start,아스테리아,"{{playerName}} 님, 이 줄은 문법 안내 시작이에요.",assets/img/goddess.png,playerName=순례자;신뢰도=2,,,,"변수 설명 보기 => intro#vars || 바로 종료하기 => intro#end",
intro,vars,아스테리아,"좋아요. `{{playerName}}`처럼 변수를 쓸 수 있어요.",assets/img/goddess.png,,,신뢰도+1,,"종료 예시 보기 => intro#end",
intro,end,아스테리아,"설명은 여기까지예요. 수고하셨어요.",assets/img/goddess.png,,,,예,,
```

---

## 7) 작성 팁

1. `태그/앵커`는 짧고 일관되게 (`intro/start`, `intro/end`).
2. 한 줄 대사는 1~2문장으로 유지.
3. `종료=예` 줄은 선택지를 비워서 마무리 의도를 분명히 표현.
4. 숨김 분기는 `히든선택지`에만 넣고, 기본 분기와 역할을 분리.
