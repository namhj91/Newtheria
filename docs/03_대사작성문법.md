# 대사 스크립트(NDS v2) 작성 문법 가이드

> 대상: 기획자/작가/비전공자  
> 인코딩: **UTF-8**

이제 대사 스크립트는 **섹션형 블록 문법(NDS v2)** 으로 작성합니다.  
기존 문법 호환은 고려하지 않습니다.

---

## 1) 핵심 구조

```text
@intro#start
[대사출력]
화자 = 아스테리아
대사 = 환영해요, 순례자.

[흐름제어]
자동넘김 = 1200

---
@intro#next
[대사출력]
화자 = 순례자
대사 = 다음 블록으로 이동했어.
```

- `@scene#anchor`: 블록 시작점
- `[섹션명]`: 항목 그룹(가독성 강화)
- `키 = 값` 또는 `키: 값`: 둘 다 허용
- `---`: 블록 구분선
- 빈 줄/`;` 주석/`//` 주석은 무시

---

## 2) 섹션 구성(권장)

### [흐름제어]
- `자동넘김`: 선택지가 없을 때 N ms 뒤 자동 진행
- `점프`: 선택지 없이 즉시 앵커 이동 (`scene#anchor`, `#anchor`, `scene`)
- `종료`: `예/아니오`, `true/false`, `on/off`
- `배경`을 비우면 직전 배경을 유지(초기 1회만 기본 배경 사용)

### [대사출력]
- `화자` (표시 이름)
- `화자포커스`/`포커스`: 스탠딩 하이라이트 키(슬롯/스탠딩 id/이름)
- `대사` (비워도 동작)

### [소리제어]
- `bgm`, `bgs`, `voice`, `sfx`
- 예시
  - `bgm = play|assets/audio/bgm.ogg|0.6|loop`
  - `sfx = assets/audio/click.ogg`
  - `voice = stop`

### [그림제어]
- `배경`, `그림`, `그림애니메이션`, `스탠딩`
- 스탠딩 확장 예시: `좌=url|이름|move(...)|auto|id=asteria_base`

### [연출제어]
- `화면효과`, `카메라`, `ui`, `진동`, `파티클`, `조명`, `씬전환`

### [변수제어]
- `변수`: `key=value;key2=value2`
- `조건`: 예) `신뢰도>=2 AND 별조각>=1`
- `효과`: 예) `신뢰도+1;별조각+=2;플래그=true`

### [선택지] / [히든선택지]
- 한 줄에 하나씩 `- 라벨 => scene#anchor`
- 선택지 메타
  - `- 비밀 이동 => intro#secret [조건:신뢰도>=2] [효과:별조각+1]`

---

## 3) 대사 없이 연출만 실행

```text
@intro#fx_only
[소리제어]
sfx = assets/audio/ui_click.ogg

[연출제어]
화면효과 = flash
카메라 = zoom(1.04,220)

[흐름제어]
자동넘김 = 600
점프 = intro#choice
```

- `대사`가 없어도 애니메이션/소리/연출은 정상 동작합니다.
- `점프`로 선택지 없이 다음 앵커로 넘어갈 수 있습니다.

---

## 4) 전체 예시

```text
@intro#choice
[대사출력]
화자 = 아스테리아
대사 = 다음 행동을 골라 주세요.

[선택지]
- 문법 설명을 듣는다 => intro#guide
- 바로 종료한다 => intro#end

[히든선택지]
- 비밀 루트 => intro#secret [조건:신뢰도>=2] [효과:별조각+1]
```

---

## 5) 문법 안전성 원칙

- 항목을 섹션으로 분리해 오타 위치를 빠르게 찾을 수 있습니다.
- `키: 값`, `키 = 값`을 모두 허용해 작성 실수를 줄였습니다.
- 대사 비우기/선택지 생략/점프 단독 사용을 공식 지원합니다.
