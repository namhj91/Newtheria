# 04. 전투 연동 규칙

- 버전: **ver.0.0.21(260329-인덱스진입페이지추가)**

## 1) 전투 진입
- 월드에서 조우/침공/토벌 발생 시 턴제 전투에 진입한다.

## 2) 전투 행위 대칭
- 플레이어가 사용할 수 있는 전투 행동(공격/스킬/아이템/후퇴)은 NPC도 동일하게 사용 가능하다.
- NPC 전용 치트 행동은 기본 규칙에서 허용하지 않는다.

## 3) 클래스리스 전투 규칙
- 고정 직업 없이 스탯/장비/스킬 조합으로 전투 스타일을 구성한다.
- 스킬 습득 조건은 직업이 아닌 대표 스탯 및 행동 이력 기반으로 판정한다.

## 4) 결과 반영
- 승패 결과는 아래 월드 상태에 반영된다.
  - 타일 치안도
  - 위협도
  - 영향권 범위
  - 정착지 자원/안정도

## 5) 장수 연동
- 장수는 전투를 통해 부상/사기/충성도 변화를 겪는다.
- 특정 장수 배치 시 정착지 또는 군단 운영 효율이 변한다.
