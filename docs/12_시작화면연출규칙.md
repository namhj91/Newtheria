# 12. 시작 화면 연출 규칙

- 버전: **ver.0.0.20(260329-시작화면HTML문서추가)**

## 1) 화면 목표
- 플레이어가 세계관 분위기를 즉시 느끼고, 주요 진입 경로를 직관적으로 선택할 수 있게 한다.

## 2) 화면 구성
1. 타이틀 로고(상단 중앙)
2. 배경 별 필드(글로우 점 파티클)
3. 카드형 메인 메뉴(중앙)
4. 우하단 버전 텍스트

## 3) 카드 선택지(4종)
- **새로운 여정**: 새로 시작
- **어떤 모험가의 일지**: 이어하기
- **대륙견문록**: 백과사전
- **신의 섭리**: 모드 설정

## 4) 인터랙션 규칙
- Hover 시 카드 글로우/살짝 확대
- 선택 시 카드가 강조되고 나머지는 디밍
- Enter/Click 확정 시 전환 연출 시작

## 5) 배경 별 연출
- 기본 상태: 느린 패럴랙스 + 은은한 글로우 점멸
- 카드 선택 시: 별들이 중심에서 바깥으로 퍼지는 확산 모션
- 연출 의도: "우주를 누비며 여정을 시작"하는 감각 제공


## 6) 별자리 라인 연출
- 임의의 글로우 점 3~6개를 연결해 "별자리" 같은 라인을 생성한다.
- 라인은 한쪽 끝에서 반대쪽으로 그려지듯 이어졌다가, 짧은 유지 후 페이드아웃된다.
- 동일 패턴 반복을 피하기 위해 연결 노드/길이/방향을 랜덤 시드로 변형한다.
- 빈도는 낮게 유지하여 과도한 시선 분산을 방지한다.

## 7) 버전 텍스트/이스터에그
- 우하단에 현재 버전 표기
- 버전 텍스트 클릭 시 이스터에그 실행:
  1. 카드가 중앙으로 모임
  2. 카드 뒷면 전환 + 셔플
  3. 다시 원래 위치로 전개
- 기능 영향: 실제 메뉴 상태/세이브 데이터에는 영향 없음

## 8) 접근성/옵션
- 모션 강도 옵션(기본/낮음/끄기)
- 화면 흔들림/과한 번쩍임 제한
- 저사양 모드에서 파티클 수 감소
- 별자리 라인 연출 빈도/밝기 옵션 제공

## 9) 사운드 연계
- Hover: 짧은 UI 톤
- 선택 확정: 상승감 있는 전환 사운드
- 셔플 이스터에그: 카드 셔플 SFX + 짧은 반짝임 SFX

## 10) 프로토타입 문서
- 루트 경로 `start_screen.html`에서 시작 화면 UI 프로토타입을 확인할 수 있다.
- 단일 HTML 문서에 배경/카드 인터랙션/버전 클릭 이스터에그를 포함한다.
