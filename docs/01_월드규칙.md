# 01. 월드 규칙

- 버전: **ver.0.0.20(260329-시작화면HTML문서추가)**

## 1) 월드 기본
- 크기: 200x200
- 좌표: (x, y, z)
- 현재 포커스: 지상계(z=0)

## 2) 계층 접근 정책
- 천상계/하계는 특정 조건에서만 진입 가능
- 기본 게임 루프에서는 지상계 중심으로 진행

## 3) 타일 필드 표준
- coord: x,y,z
- terrainType
- manaSaturation
- security
- nationId
- settlementId
- influenceId
- sparkleLight
- specialProduct
- threatInfo
- specialTileType

## 4) 지상계 지형 목록
- 평원
- 숲
- 고대림
- 구릉지
- 협곡
- 산맥
- 화산지대
- 만년설산
- 습지
- 황무지
- 사막
- 모래해변
- 강
- 호수
- 해안
- 바다
- 심해

## 5) 종족 상징 특수 타일
### 5.1 세계수 (엘프족)
- 숲/고대림 지역에서만 생성
- 세계수 중심 타일의 인접 8방향은 숲 계열 타일이어야 함

### 5.2 드래곤 둥지 (용인족)
- 월드의 최고 고도 타일에 생성
- 동률인 경우 산맥 > 만년설산 > 협곡 순으로 우선 배치

### 5.3 고대 성소 (인간족)
- 최초 성소 건립 도시가 파괴되어 폐허가 된 시점에 해당 폐허 타일로 변환 생성
- 자연 생성이 아닌 이벤트 기반 생성

### 5.4 미스릴 광산 (드워프족)
- 산맥 타일에서만 생성
- 드래곤 둥지와 최소 거리 `D_min` 이상 유지
- 거리 계산은 타일 맨해튼 거리 또는 헥스 거리(맵 구조 확정 후 택1)

## 6) 생성 우선순위(초안)
1. 드래곤 둥지 위치 확정
2. 미스릴 광산 후보 중 거리 조건을 만족하는 타일 확정
3. 세계수 숲권역 배치
4. 고대 성소는 게임 진행 이벤트로 생성
