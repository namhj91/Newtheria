# 99. 변경 이력

## ver.0.0.44(260331-시작화면스크립트모듈화)
- 시작 화면 스크립트를 `layout`/`effects`/`reroll`/`drag`/`bindEvents` 단위로 재구성해 역할별 책임을 분리
- 기존 카드 선택/드래그/셔플/리롤/별 배경 동작은 동일하게 유지하면서 코드 가독성과 유지보수성을 개선
- 시작 화면 버전 버튼 표기를 `ver.0.0.44(260331-시작화면스크립트모듈화)`로 갱신

## ver.0.0.43(260331-시작화면파일분리상수화)
- 시작 화면의 인라인 `<style>`/`<script>`를 각각 `assets/css/start_screen.css`, `assets/js/start_screen.js`로 분리
- 시작 화면 동작 관련 매직 넘버를 `UI` 상수 객체로 정리해 유지보수성을 개선(동작/타이밍 값은 기존과 동일 유지)
- 시작 화면 버전 버튼 표기를 `ver.0.0.43(260331-시작화면파일분리상수화)`로 갱신

## ver.0.0.38(260331-카드뒤집힘플래튼수정)
- 카드 뒤집기 축인 `.card-fan-card-inner`에서 `overflow: hidden`/`border-radius`를 제거해 3D 플래튼(flatten) 현상을 방지
- 카드 페이스(`.card-fan-card-face`)에 `overflow: hidden`을 적용해 모서리 클리핑은 유지하면서 뒷면 렌더링이 정상 노출되도록 조정
- 시작 화면 버전 버튼 표기를 `ver.0.0.38(260331-카드뒤집힘플래튼수정)`로 갱신

## ver.0.0.37(260331-카드뒷면에테리아디자인)
- 카드 뒷면이 앞면 반전처럼 보이던 문제를 완화하기 위해 카드 페이스에 `-webkit-backface-visibility: hidden`을 추가
- 카드 뒷면 디자인을 시질 아이콘(✦) + `AetheriA` 워드마크 중심의 테마 스타일로 교체
- 시작 화면 버전 버튼 표기를 `ver.0.0.37(260331-카드뒷면에테리아디자인)`로 갱신

## ver.0.0.36(260331-카드전체뒤집힘수정)
- 카드 외곽 프레임(배경/테두리/그림자) 스타일을 고정 컨테이너(`.card-fan-card`)에서 뒤집힘 레이어(`.card-fan-card-inner`, `.card-fan-card-face`)로 이동
- 리롤 뒤집기 시 카드 안쪽 면만 회전하던 문제를 수정해 카드 전체가 뒤집히는 것처럼 보이도록 개선
- 시작 화면 버전 버튼 표기를 `ver.0.0.36(260331-카드전체뒤집힘수정)`로 갱신

## ver.0.0.35(260331-카드셔플회전뒤집힘수정)
- 시작 화면 카드 내부 스핀/뒤집힘 레이어(`.card-fan-card-spin`, `.card-fan-card-inner`)에 `display: block`을 추가해 transform이 실제 적용되도록 수정
- 리롤 시 중앙 수렴 이후 카드 뒷면 뒤집힘과 카드별 회전 연출이 보이지 않던 원인을 해결
- 시작 화면 버전 버튼 표기를 `ver.0.0.35(260331-카드셔플회전뒤집힘수정)`로 갱신

## ver.0.0.34(260331-카드뒷면셔플전반전)

- 시작 화면 버전 버튼 표기를 `ver.0.0.34(260331-카드뒷면셔플전반전)`로 갱신
