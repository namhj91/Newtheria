대화 테스트 설정

1) 데이터 소스
2) 시작 위치
3) 배경 설정
4) 변수/추적 (디버그)
5) 문법 힌트

헤더형 .asdf 문법을 사용합니다.
<event:이름> / <scene#n:이름> / <block:anchor>를 사용하고, 블록 종료는 --- 입니다.
블록 내부는 { ... }로 감싸고 key = value만 허용합니다.
선택지는 choice = 라벨 -> anchor를 사용하고, 히든 선택지는 hidden_choice = 라벨 -> anchor로 작성합니다. (일반 선택지를 모두 버리면 히든 선택지 노출)
문자열은 {player_name} 같은 변수 치환을 지원하고, type_speed = 24 / wait = 800으로 타자기·자동진행 연출을 줄 수 있습니다.
인라인 효과로 [[spoiler:내용]]/[[스포:내용]], [[glitch:내용]]/[[깨진글:내용]]을 사용할 수 있으며 line/speaker 모두 적용됩니다.
particle = snow|ember|off, particle_count, particle_duration으로 파티클 연출을 제어할 수 있습니다.
screen_flash = 220, screen_blackout = 700으로 지속시간을 주고, screen_flash = 220,120, screen_blackout = 900,300처럼 한 키에 변화시간까지 함께 줄 수 있습니다. (*_fade 분리 키도 계속 지원, 단독 지정 시 기본 지속시간으로 동작)
skip_lock = on|off(또는 skip = lock|unlock)으로 스킵 잠금을 제어할 수 있으며, 해제 지시가 없으면 scene 전환 전까지 유지됩니다.
씬 헤더 직후 { actor / slot_left / slot_right / focus }로 기본 스탠딩을 정의하며, actor = id는 (1) scene 번들 → (2) 활성 세이브 슬롯 캐릭터 → (3) 대표 캐릭터 localStorage 순서로 조회됩니다.
진행 포인터는 같은 탭에서 sessionStorage(newtheria.dialogue.progress)로 복원됩니다.
