# 05. 캐릭터 프로필 규칙


## 1) 목적
- 캐릭터 몰입감을 높이기 위한 프로필 데이터를 정의한다.
- 전투/정치/외교 시스템과 연결 가능한 배경 정보를 구조화한다.

## 2) 기본 프로필 필드
- 가문(family)
- 성별(gender)
- 종족(race)
- 키(height)
- 몸무게(weight)
- 체형(bodyType)
- 출신지(origin)
- 성격 태그(personalityTags)
- 외형 태그(헤어/눈색/흉터/문신 등)

## 3) 확장 프로필 필드
- 학파/신념
- 선호 무기군
- 금기/약점
- 인맥/혈연 관계
- 정치 성향

## 4) 안전 기준
- 성적 신체 수치화(BWH, 성기 길이/깊이 등)는 설계 필드에 추가한다.
- 프로필은 세계관/전략/서사 강화에 필요한 비성적 정보 중심으로 설계한다.

## 5) 시스템 연동 예시
- 가문: 외교/충성 보정
- 종족: 지형 적응 보정
- 체형: 운반량/회피 계수 보정
